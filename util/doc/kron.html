void kron

(long* a, long m, long* b, long n, long* c)

Find kronecker product of m by 1 vector a and n by 1 vector b.

Documentation

Find kronecker product of m by 1 vector a and n by 1 vector b. Resulting m*n by 1 product is passed back via c
Parameters:
a - an m x 1 vector
m - length of a
b - an n x 1 vector
n - length of b
c - an m x n array which, on exit, holds the kronecker product
